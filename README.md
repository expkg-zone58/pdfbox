# Pdfbox
BaseX (10+) interface to [Pdfbox](https://pdfbox.apache.org/) version 3

## Features
* read page count
* read outline
* read pagelabels
* read page text
* save pdf page range to new pdf
* save pdf page as image


## Jars
* fontbox-3.0.2.jar
* pdfbox-3.0.2.jar
* pdfbox-io-3.0.2.jar
* commons-logging-1.3.1.jar
